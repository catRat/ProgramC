	printf("This is a C program.\n");

例1.2 求两个整数之和。

解题思路：设置3个变量，a和b用来存放两个整数，sum用来存放和数。用赋值运算符“=”把相加的结果传送给sum。

编写程序：
\begin{lstlisting}
#include<stdio.h>
int main()
{
	int a, b, sum;
	a = 123;
	b = 456;
	sum = a + b;
	printf("sum is %d\n", sum);
	return 0;
}
\end{lstlisting}
